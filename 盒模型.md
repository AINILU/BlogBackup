### box-sizing

 - border-box
 - content-box(default)

### width/min-width/max-width
- content box宽度
- 百分数相对于父元素的content box宽度

### height/min-height/max-height
- 只有当父元素的高度不依赖于该元素时，百分比才生效

### margin

- 属性具有重叠的特性，顺序很重要

### border

- border-width
- border-style
- border-color

border三角形
```
.box {
    width: 0;
    border-width: 200px;        
    border-style: solid;
    border-color:  rgba(255, 0, 0, 0) rgba(255, 0, 0, 0) rgba(255, 0, 0, 0) #f35;
}
```
### overflow

- overflow: scroll;
- overflow: hidden;
- overflow: auto;

### 块级元素与行级元素
- 行级margin-top、margin-bottom不起作用
- 行级padding-top、padding-bottom不影响布局
